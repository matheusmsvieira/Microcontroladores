1.Quais as diferenças entre os barramentos de dados e de endereços?
	O barramentos de dados é um canal de transmissão que transporta dados e informações importantes entre o microprocessador e a memória no interior de um computador.
Já o barramento de endereços é uma via de transmissão usada pelo microprocessador para acessar os endereços de um memória na qual desejar acessar.

2.Quais são as diferenças entre as memórias RAM e ROM?
	A memória RAM (Random Access Memory) é uma memória de leitura e escrita, o conteúdo pode ser lido e alterado durante seu uso no circuito. É uma memória volátil, ou seja, os dados armazenados serão perdidos quando o fornecimento de energia for interrompido.
A memória ROM (Read-only memory) é uma memória somente para leitura. O conteúdo pode ser lido durante o uso no circutio e a gravação é feita fora da operação normal do circuito.

4. Quais são as diferenças, vantagens e desvantagens das arquiteturas Harvard e Von Neumann?
	A diferença entre a arquitetura Von Neunmann e a Harvard é que a última separa o armazenamento e o comportamento das instruções do CPU e os dados, enquanto a anterior utiliza o mesmo espaço de memória para ambos. A arquitetura  de Von Neumann é mais simples porém é mais lento pois não permite acesso simultâneo às memórias e é geralmente CISC. Já a arquitetura Harvard é mais complexa no entanto é mais rápido pois permite o acesso simultâneo às memórias, pipelining.
